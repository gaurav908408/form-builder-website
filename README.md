# 📝 Form Builder Application

A full-stack MERN Form Builder Application that allows users to create, manage, and share custom forms with ease. Users can register, log in, build dynamic forms, share them using a unique link, collect responses, and view all submitted responses from a personalized dashboard.

The application is built using the MERN stack with a responsive and user-friendly interface, making it suitable for surveys, feedback forms, registration forms, quizzes, and many other use cases.

---

# ✨ Features

- 🔐 User Authentication (Register & Login)
- 👤 Secure User Dashboard
- ➕ Create Dynamic Forms
- ✏️ Edit Existing Forms
- 🗑️ Delete Forms
- 📋 Add Multiple Field Types
  - Text Field
  - Email Field
  - Number Field
  - Textarea
  - Radio Button
  - Checkbox
- 🔗 Generate Shareable Form Link
- 🌐 Public Form Access
- 📥 Submit Form Responses
- 📊 View Submitted Responses
- 📱 Fully Responsive Design
- ⚡ Fast and Modern User Interface
- 🔒 JWT Authentication
- ☁️ Backend Deployed on Render
- ▲ Frontend Deployed on Vercel

# 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/gaurav908408/form-builder-website.git
```

### Move into the project

```bash
cd form-builder-website
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Install Frontend Dependencies

```bash
cd ../frontend
npm install
```

### Start Backend

```bash
npm run dev
```

### Start Frontend

```bash
npm run dev
```
