# Form Builder Website

A full-stack Form Builder application where users can create, share, and manage custom forms.

## Features

- User Authentication (Login & Register)
- Create Custom Forms
- Add Text, Email, Number, Radio & Checkbox Fields
- Share Form with Link
- Submit Responses
- View Submitted Responses
- Responsive UI

## Tech Stack

### Frontend
- Next.js
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Project Structure

```
form-builder-website/
├── backend/
└── frontend/
```

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
